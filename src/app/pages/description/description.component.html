Known issues

On certain view dimensions, you can see that full-width alignment of the latest row looks not appropriate.
In the real application, I would suggest finding a proper way of using images (by changing the technical task), or either create an auto-alignment function where you will check the number of items fit the row and depends on this set row/column parameters. But it sounds quite "hacky", so I decided to follow the initial task and solve it with css.
